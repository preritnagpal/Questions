Question Project

1. What is HTML ?

HTML, which stands for HyperText Markup Language, is the standard language used to create and design the structure of web pages. It uses a system of tags to define elements such as headings, paragraphs, links, images, and other types of content.

2. What is CSS ?

CSS, or Cascading Style Sheets, is a stylesheet language used to control the presentation of HTML documents. It allows you to apply styles like colors, fonts, and layout to web pages, enhancing their appearance and making them responsive to different devices and screen sizes.

3. What is JS ?

JavaScript (JS) is a versatile programming language used to create dynamic and interactive features on websites.JS runs in the browser, allowing for real-time updates without refreshing the page, enhancing user experience and engagement.
